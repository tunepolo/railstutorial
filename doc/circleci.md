# CircleCI

## ローカルでの動かし方。

参考リンクの手順でDocker/CircleCI CLIをインストールしておく。

.circleci/config.ymlに設定を記載し、下記コマンドで設定ファイルの検証を行う。

```
$ circleci config validate
Config file at .circleci/config.yml is valid.
```
